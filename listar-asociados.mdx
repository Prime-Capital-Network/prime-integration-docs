Obtén una lista paginada de asociados en tu empresa.

```http
GET /associates
```

<Note>
  Este endpoint requiere el scope **`Associates.Read`**.
</Note>

### Query parameters

<ParamField query="status_id" type="byte">
  Filtrar por el estatus de sociado. Valores aceptados:

  - 1 (En Aprobación)
  - 2 (Activo)
  - 3 (Inactivo)
  - 4 (Suspendido)
  - 5 (Cancelado)
</ParamField>

<ParamField query="skip" default="20" type="integer">
  Número de registros a omitir desde el inicio de los resultados. Utilízado junto con `take` para paginar los resultados.
</ParamField>

<ParamField query="take" type="integer">
  Número de registros a retornar. Por defecto `20`, máximo `100`.
</ParamField>

<ParamField query="name" type="StringSearchFilter">
  Filtra por nombre del asociado.
</ParamField>

<ParamField query="alias" type="StringSearchFilter">
  Filtra por alias o nombre corto del asociado.
</ParamField>

<ParamField query="IdentityDocument.Mask" type="StringSearchFilter">
  Filtra por la máscara del tipo de documento de identidad.
</ParamField>

<ParamField query="IdentityDocument.Number" type="StringSearchFilter">
  Filtra por el número de documento de identidad.
</ParamField>

<ParamField query="countryId" type="StringSearchFilter">
  Filtra por el identificador del país del asociado.
</ParamField>

<ParamField query="cityName" type="StringSearchFilter">
  Filtra por el nombre de la ciudad del asociado.
</ParamField>

<ParamField query="isCustomer" type="BooleanSearchFilter">
  Filtra asociados que operan como cliente.
</ParamField>

<ParamField query="isVendor" type="BooleanSearchFilter">
  Filtra asociados que operan como suplidor.
</ParamField>

<ParamField query="isAgent" type="BooleanSearchFilter">
  Filtra asociados que operan como agente.
</ParamField>

<ParamField query="isCustomAgent" type="BooleanSearchFilter">
  Filtra asociados que operan como agente aduanal.
</ParamField>

<ParamField query="isWarehouse" type="BooleanSearchFilter">
  Filtra asociados que operan como almacén.
</ParamField>

<ParamField query="isSealine" type="BooleanSearchFilter">
  Filtra asociados que operan como línea marítima.
</ParamField>

<ParamField query="isAirline" type="BooleanSearchFilter">
  Filtra asociados que operan como aerolínea.
</ParamField>

<ParamField query="isTrucking" type="BooleanSearchFilter">
  Filtra asociados que operan como transportista.
</ParamField>

<ParamField query="isBank" type="BooleanSearchFilter">
  Filtra asociados que operan como entidad bancaria.
</ParamField>

<ParamField query="isInsurance" type="BooleanSearchFilter">
  Filtra asociados que operan como aseguradora.
</ParamField>

<ParamField query="isEmployee" type="BooleanSearchFilter">
  Filtra asociados que son empleados internos.
</ParamField>

<ParamField query="salesPersonName" type="StringSearchFilter">
  Filtra por el nombre del vendedor asignado.
</ParamField>

<ParamField query="accountExecutiveName" type="StringSearchFilter">
  Filtra por el nombre del ejecutivo de cuenta asignado.
</ParamField>

<ParamField query="debtCollectorName" type="StringSearchFilter">
  Filtra por el nombre del gestor de cobros asignado.
</ParamField>

<ParamField query="businessTypeName" type="StringSearchFilter">
  Filtra por el nombre del tipo de negocio.
</ParamField>

<ParamField query="commercialActivityName" type="StringSearchFilter">
  Filtra por el nombre de la actividad comercial.
</ParamField>

<ParamField query="CustomerPaymentTerm.Name" type="StringSearchFilter">
  Filtra por el nombre del término de pago del cliente.
</ParamField>

<ParamField query="VendorPaymentTerm.Name" type="StringSearchFilter">
  Filtra por el nombre del término de pago del suplidor.
</ParamField>

<ParamField query="hasPassedDueDiligence" type="BooleanSearchFilter">
  Filtra asociados que completaron el proceso de debida diligencia.
</ParamField>

<ParamField query="statusName" type="StringSearchFilter">
  Filtra por el nombre del estado del asociado.
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request GET \
  --url "https://api.primenetcloud.com/v1/associates?status_id=2&skip=20&take=20" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID"
```

```javascript JavaScript
const response = await fetch(
  "https://api.primenetcloud.com/v1/associates?status_id=2&skip=20&take=20",
  {
    method: "GET",
    headers: {
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "TENANT_ID"
    },
  }
);

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.primenetcloud.com/v1/associates",
    params={
        "status_id": 2,
        "skip": 20,
        "take": 20,
    },
   headers={
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "TENANT_ID"
    }
)

print(response.json())
```

</CodeGroup>
