---
title: "Actualizar Asociado"
sidebarTitle: "Actualizar"
description: "Este endpoint permite actualizar los datos de un asociado existente en la empresa."
---

```http
PUT /associates/{id}
```

<Note>
  Este endpoint requiere el scope **`Associates.Write`**.
</Note>

<Warning>
  Este es un reemplazo completo del recurso (PUT). Todos los campos enviados en el body sobrescribirán el estado actual del asociado. Para cambios parciales, verificar la sección [Actualizar Parcial Asociado](/api-reference/associates/actualizar-parcial-asociado).
</Warning>

### Path parameters

<ParamField path="id" type="integer" required>
  Identificador único del asociado a actualizar.
</ParamField>

### Body parameters

<ParamField body="name" type="string" required>
  Nombre completo del asociado.
</ParamField>

<ParamField body="alias" type="string">
  Nombre corto o alias utilizado internamente.
</ParamField>

<ParamField body="typeId" type="integer" required>
  Identificador del tipo de asociado.
</ParamField>

<ParamField body="identityDocumentId" type="integer" required>
  Identificador del tipo de documento de identidad.
</ParamField>

<ParamField body="identityDocumentNumber" type="string">
  Número de documento de identidad. Debe coincidir con el largo definido por el `identityDocumentId` seleccionado.
</ParamField>

<ParamField body="franchiseId" type="integer">
  Identificador de la franquicia a la que pertenece el asociado.
</ParamField>

<ParamField body="addressLine" type="string">
  Dirección física del asociado.
</ParamField>

<ParamField body="countryId" type="string">
  Identificador del país del asociado.
</ParamField>

<ParamField body="cityId" type="string">
  Identificador de la ciudad del asociado.
</ParamField>

<ParamField body="sectorId" type="integer">
  Identificador del sector o barrio del asociado.
</ParamField>

<ParamField body="latitude" type="number">
  Latitud geográfica. Valor entre `-90` y `90`.
</ParamField>

<ParamField body="longitude" type="number">
  Longitud geográfica. Valor entre `-180` y `180`.
</ParamField>

<ParamField body="postcode" type="string">
  Código postal del asociado.
</ParamField>

<ParamField body="isCustomer" type="boolean">
  Indica si el asociado opera como cliente.
</ParamField>

<ParamField body="isVendor" type="boolean">
  Indica si el asociado opera como suplidor.
</ParamField>

<ParamField body="isAgent" type="boolean">
  Indica si el asociado opera como agente.
</ParamField>

<ParamField body="isCustomAgent" type="boolean">
  Indica si el asociado opera como agente aduanal.
</ParamField>

<ParamField body="isWarehouse" type="boolean">
  Indica si el asociado opera como almacén.
</ParamField>

<ParamField body="isSealine" type="boolean">
  Indica si el asociado opera como línea marítima.
</ParamField>

<ParamField body="isAirline" type="boolean">
  Indica si el asociado opera como aerolínea.
</ParamField>

<ParamField body="isTrucking" type="boolean">
  Indica si el asociado opera como transportista.
</ParamField>

<ParamField body="isBank" type="boolean">
  Indica si el asociado opera como entidad bancaria.
</ParamField>

<ParamField body="isInsurance" type="boolean">
  Indica si el asociado opera como aseguradora.
</ParamField>

<ParamField body="isEmployee" type="boolean">
  Indica si el asociado es un empleado interno.
</ParamField>

<ParamField body="customerAccountGroupId" type="integer">
  Identificador del grupo contable del cliente. Requerido si `isCustomer` es `true` . Su tipo de asociado `AssociateType` debe ser del tipo **Customer.**
</ParamField>

<ParamField body="customerPaymentTermId" type="integer">
  Identificador del término de pago del cliente.  Requerido si `isCustomer` es `true` .
</ParamField>

<ParamField body="businessTypeId" type="integer">
  Identificador del tipo de negocio del cliente.  Requerido si `isCustomer` es `true` .
</ParamField>

<ParamField body="customerCurrencyId" type="string">
  Código de la moneda del cliente (ej. `DOP`, `USD`).  Requerido si `isCustomer` es `true` .
</ParamField>

<ParamField body="customerBillingCreditLimit" type="number">
  Límite de crédito de facturación del cliente.
</ParamField>

<ParamField body="customerReimbursementCreditLimit" type="number">
  Límite de crédito de reembolso del cliente.
</ParamField>

<ParamField body="vendorAccountGroupId" type="integer">
  Identificador del grupo contable del suplidor.  Requerido si `isVendor` es `true` . Su tipo de asociado `AssociateType` debe ser del tipo **Vendor.**
</ParamField>

<ParamField body="vendorPaymentTermId" type="integer">
  Identificador del término de pago del suplidor.  Requerido si `isVendor` es `true` .
</ParamField>

<ParamField body="vendorCurrencyId" type="string">
  Código de la moneda del suplidor (ej. `DOP`, `USD`).  Requerido si `isVendor` es `true` .
</ParamField>

<ParamField body="vendorGoodServiceId" type="string">
  Identificador del bien o servicio principal del suplidor.
</ParamField>

<ParamField body="vendorBusinessTypeId" type="integer">
  Identificador del tipo de negocio del suplidor.
</ParamField>

<ParamField body="isVendorNCFRequired" type="boolean">
  Indica si el suplidor está obligado a emitir NCF. Por defecto, `false`
</ParamField>

<ParamField body="salesTaxRetentionId" type="integer">
  Identificador del tipo de retención de ITBIS aplicable.
</ParamField>

<ParamField body="incomeTaxRetentionId" type="integer">
  Identificador del tipo de retención de ISR aplicable.
</ParamField>

<ParamField body="isRetentionRequired" type="boolean">
  Indica si aplica retención de impuestos al asociado. Por defecto, `false`
</ParamField>

<ParamField body="customRegimeId" type="integer">
  Identificador del régimen aduanal del asociado. `null` si no aplica.
</ParamField>

<ParamField body="salesPersonId" type="integer">
  Identificador del vendedor asignado al asociado.
</ParamField>

<ParamField body="accountExecutiveId" type="integer">
  Identificador del ejecutivo de cuenta asignado.
</ParamField>

<ParamField body="debtCollectorId" type="integer">
  Identificador del gestor de cobros asignado.
</ParamField>

<ParamField body="commercialActivityId" type="integer">
  Identificador de la actividad comercial del asociado.
</ParamField>

<ParamField body="departmentId" type="integer">
  Identificador del departamento asignado. Aplica si `isEmployee` es `true`.
</ParamField>

<ParamField body="costCenterId" type="integer">
  Identificador del centro de costo asignado.
</ParamField>

<ParamField body="projectId" type="integer">
  Identificador del proyecto asignado.
</ParamField>

<ParamField body="standardCarrierAlphaCode" type="string">
  Código SCAC del transportista. Máximo 4 caracteres.
</ParamField>

<ParamField body="agentCode" type="string">
  Código interno del agente.
</ParamField>

<ParamField body="iataCode" type="string">
  Código IATA de la aerolínea. Máximo 4 caracteres.
</ParamField>

<ParamField body="shouldHoldShipment" type="boolean">
  Indica si los embarques del asociado deben ser retenidos. Por defecto, `false`
</ParamField>

<ParamField body="shouldHoldCustom" type="boolean">
  Indica si las gestiones aduanales del asociado deben ser retenidas. Por defecto, `false`
</ParamField>

<ParamField body="shouldNotifyArrivalStatement" type="boolean">
  Indica si el asociado debe recibir notificaciones de aviso de llegada. Por defecto, `false`
</ParamField>

<ParamField body="attention" type="string">
  Nombre de la persona de contacto principal.
</ParamField>

<ParamField body="phone" type="string">
  Número de teléfono del asociado.
</ParamField>

<ParamField body="mail" type="string">
  Correo electrónico del asociado.
</ParamField>

<ParamField body="website" type="string">
  Sitio web del asociado.
</ParamField>

<ParamField body="referenceCode" type="string">
  Código de referencia externo del asociado.
</ParamField>

<ParamField body="consignorType" type="string">
  Tipo de consignante. Valores aceptados: `Personal`, `Business`.
</ParamField>

<ParamField body="isProfit" type="boolean">
  Indica si el asociado es una entidad con fines de lucro. Por defecto, `false`
</ParamField>

<ParamField body="quotePrintFormat" type="string">
  Idioma del formato de impresión de cotizaciones. Valores aceptados: `Spanish`, `English`.
</ParamField>

<ParamField body="invoicePrintFormat" type="string">
  Idioma del formato de impresión de facturas. Valores aceptados: `Spanish`, `English`.
</ParamField>

<ParamField body="reimbursementPrintFormat" type="string">
  Idioma del formato de impresión de reembolsos. Valores aceptados: `Spanish`, `English`, `ReimbursementInvoice` y `InternationalFreightDocument`.
</ParamField>

<ParamField body="hasPassedDueDiligence" type="boolean">
  Indica si el asociado completó el proceso de debida diligencia. Por defecto, `false`
</ParamField>

<ParamField body="statusId" type="integer" required>
  Identificador del estado inicial del asociado.
</ParamField>

### Response

Si la actualización es exitosa, el endpoint retorna `204 No Content` sin body.

### Errores

<ResponseField name="404 Not Found">
  No existe ningún asociado con el `id` especificado en la URL.
</ResponseField>

<ResponseField name="403 Forbidden">
  El API Key no tiene el scope `Associates.Write` requerido.
</ResponseField>
