---

Obtén una asociado por ID en la empresa.

```http
GET /associates/{id}
```

<Note>
  Este endpoint requiere el scope **`Associates.Read`**.
</Note>

### Path parameters

<ParamField path="id" type="integer" required>
  Identificador único del asociado.
</ParamField>

### Query parameters

<ParamField query="include" type="string">
  Incluye los detalles a traer en el mismo request. Valores aceptados: `Addresses`, `Categories`, `Documents`, `Services`, `Contacts`, y `Tariffs`. De no ser enviados, el API devuelve un array vacío.
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request GET \
  --url "https://api.primenetcloud.com/v1/associates/1023?include=Documents" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID"
```

```javascript JavaScript
const response = await fetch(
  "https://api.primenetcloud.com/v1/associates/1023?include=Documents",
  {
    method: "GET",
    headers: {
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "e3d5e6af-3ec2-49c1-92bd-9632171e9a66"
    },
  }
);
const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.primenetcloud.com/v1/associates/1023",
    params={"include": "Documents"},
    headers={
      "X-Api-Key": "pnk_abc123.s3cr3tValue",
      "X-Tenant-Id": "e3d5e6af-3ec2-49c1-92bd-9632171e9a66"
    }
)
print(response.json())
```

</CodeGroup>
