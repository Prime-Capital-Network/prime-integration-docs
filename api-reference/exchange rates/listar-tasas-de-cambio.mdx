Obtén una lista limitada de tasas de cambio de 2 monedas.

```http
GET /currencies/rates
```

<Note>
  Este endpoint requiere el scope **`ExchangeRates.Read`**.
</Note>

### Query parameters

<ParamField query="BaseId" type="string" required>
  ID de la moneda base.
</ParamField>

<ParamField query="ConvertId" type="BooleanSearchFilter" required>
  ID de la moneda convertida.
</ParamField>

<ParamField query="Count" type="integer">
  Cantidad a traer. Valor entre `1` y `20`. Por defecto, `5`.
</ParamField>

<ParamField query="Date" type="datetimeoffset">
  Fecha límite de búsqueda.
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request GET \
  --url "https://api.primenetcloud.com/v1/currencies/rates?BaseId=USD&ConvertId=DOP&Count=5" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID"
```

```javascript JavaScript
const response = await fetch(
  "https://api.primenetcloud.com/v1/currencies/rates?BaseId=USD&ConvertId=DOP&Count=5",
  {
    method: "GET",
    headers: {
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "TENANT_ID",
    },
  }
);

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.primenetcloud.com/v1/currencies/rates",
    params={
        "BaseId": "USD",
        "ConvertId": "DOP",
        "Count": 5,
    },
    headers={
        "X-Api-Key": "YOUR_API_KEY",
        "X-Tenant-Id": "TENANT_ID",
    },
)

print(response.json())
```

</CodeGroup>

### Response fields

<ResponseField name="id" type="integer">
  Identificador único del representante.
</ResponseField>

<ResponseField name="name" type="string">
  Nombre completo del representante.
</ResponseField>

<ResponseField name="isSalesPerson" type="boolean">
  Indica si el representante es un vendedor.
</ResponseField>

<ResponseField name="isExecutiveAccount" type="boolean">
  Indica si el representante es un ejecutivo de cuentas.
</ResponseField>

<ResponseField name="isCustomerService" type="boolean">
  Indica si el representante pertenece al área de servicio al cliente.
</ResponseField>

<ResponseField name="isDebtCollector" type="boolean">
  Indica si el representante es un gestor de cobros o cobrador.
</ResponseField>

<ResponseField name="isActive" type="boolean">
  Indica si el representante está actualmente activo en el sistema.
</ResponseField>
