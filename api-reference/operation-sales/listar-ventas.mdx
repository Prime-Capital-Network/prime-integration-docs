title: "Listar"
---

Obtén la lista de estimados de venta (operation sales) asociados a una operación logística específica en la empresa.

```http
GET /operation-sales
```

<Note>
  Este endpoint requiere el scope **`OperationSales.Read`**.
</Note>

### Query parameters

<ParamField query="operation_id" type="integer" required>
  Identificador de la operación logística de la cual se quieren obtener los estimados de venta.
</ParamField>

<ParamField query="category" type="string" required>
  Categoría de la operación logística. Valores aceptados:

  - `Shipment` (Embarque)
  - `CustomManagement` (Aduanas)
  - `Additional` (Misceláneos)
  - `InlandFreight` (Transporte Terrestre)
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request GET \
  --url "https://api.primenetcloud.com/v1/operation-sales?operation_id=5321&category=Shipment" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID"
```

```javascript JavaScript
const response = await fetch(
  "https://api.primenetcloud.com/v1/operation-sales?operation_id=5321&category=Shipment",
  {
    method: "GET",
    headers: {
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "TENANT_ID"
    },
  }
);

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.primenetcloud.com/v1/operation-sales",
    params={
        "operation_id": 5321,
        "category": "Shipment",
    },
   headers={
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "TENANT_ID"
    }
)

print(response.json())
```

</CodeGroup>

### Response fields

<ResponseField name="id" type="integer">
  Identificador único del estimado de venta.
</ResponseField>

<ResponseField name="charge" type="object">
  Cargo asociado a la venta.

  <Expandable title="propiedades">
    <ResponseField name="id" type="integer">
      Identificador del cargo.
    </ResponseField>

    <ResponseField name="name" type="string">
      Nombre del cargo.
    </ResponseField>

    <ResponseField name="isUsedForSales" type="boolean">
      Indica si el cargo se utiliza para ventas.
    </ResponseField>

    <ResponseField name="isUsedForCosts" type="boolean">
      Indica si el cargo se utiliza para costos.
    </ResponseField>

    <ResponseField name="appliesForReimbursement" type="boolean">
      Indica si el cargo aplica para reembolso.
    </ResponseField>

    <ResponseField name="taxRate" type="decimal">
      Tasa de impuesto del cargo.
    </ResponseField>

    <ResponseField name="hasFixedTax" type="boolean">
      Indica si el cargo tiene un impuesto fijo.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="chargeDescription" type="string">
