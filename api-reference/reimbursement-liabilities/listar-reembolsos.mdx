---
title: "Listar"
---

Obtén una lista paginada de reembolsos (cuentas por pagar) en la empresa.

```http
GET /reimbursement-liabilities
```

<Note>
  Este endpoint requiere el scope **`ReimbursementLiabilities.Read`**.
</Note>

### Query parameters

<ParamField query="start" type="datetime" required>
  Fecha de inicio del rango de búsqueda.
</ParamField>

<ParamField query="end" type="datetime" required>
  Fecha de fin del rango de búsqueda.
</ParamField>

<ParamField query="sort_by" type="string">
  Campo por el cual ordenar los resultados. Valores aceptados:

  - `Id`
  - `Number`
  - `JournalNumber`
  - `TransactionNumber`
  - `Date`
  - `Vendor.Name`
  - `IdentityDocument.Mask`
  - `IdentityDocumentNumber`
  - `Reference`
  - `Currency.Id`
  - `Operation.Id`
  - `Operation.TransactionNumber`
  - `Master.Id`
  - `Master.TransactionNumber`
  - `TotalAmount`
  - `InsertedBy`
  - `Status`
</ParamField>

<ParamField query="skip" default="20" type="integer">
  Número de registros a omitir desde el inicio de los resultados. Utilízado junto con `take` para paginar los resultados.
</ParamField>

<ParamField query="take" type="integer">
  Número de registros a retornar. Por defecto `20`, máximo `100`.
</ParamField>

<ParamField query="Criteria" type="string">
  Filtra los reembolsos según el estatus de su comprobante contable.

  Valores aceptados:
  - Active
  - New
  - Posted
  - Cancelled
  - Deleted
</ParamField>

<ParamField query="Number" type="StringSearchFilter">
  Filtra por el número del reembolso.
</ParamField>

<ParamField query="JournalNumber" type="RangeSearchFilter<integer>">
  Filtra por el número de comprobante contable del reembolso.
</ParamField>

<ParamField query="TransactionNumber" type="RangeSearchFilter<integer>">
  Filtra por el número de transacción del reembolso.
</ParamField>

<ParamField query="Vendor.Name" type="StringSearchFilter">
  Filtra por el nombre del vendor.
</ParamField>

<ParamField query="IdentityDocument.Mask" type="StringSearchFilter">
  Filtra por la máscara del documento de identidad del vendor.
</ParamField>

<ParamField query="IdentityDocumentNumber" type="StringSearchFilter">
  Filtra por el número del documento de identidad del vendor.
</ParamField>

<ParamField query="Reference" type="StringSearchFilter">
  Filtra por la referencia del reembolso.
</ParamField>

<ParamField query="Currency.Id" type="StringSearchFilter">
  Filtra por el identificador de la moneda.
</ParamField>

<ParamField query="Operation.Id" type="RangeSearchFilter<integer>">
  Filtra por el identificador de la operación logística asociada.
</ParamField>

<ParamField query="Operation.TransactionNumber" type="RangeSearchFilter<integer>">
  Filtra por el número de transacción de la operación logística asociada.
</ParamField>

<ParamField query="Operation.Category.Id" type="EnumSearchFilter">
  Filtra por la categoría de la operación logística asociada. Valores aceptados:

  - `Shipment`
  - `CustomManagement`
  - `Additional`
  - `InlandFreight`
</ParamField>

<ParamField query="Master.Id" type="RangeSearchFilter<integer>">
  Filtra por el identificador del master asociado.
</ParamField>

<ParamField query="Master.TransactionNumber" type="RangeSearchFilter<integer>">
  Filtra por el número de transacción del master asociado.
</ParamField>

<ParamField query="TotalDebit" type="RangeSearchFilter<decimal>">
  Filtra por el total del débito del reembolso.
</ParamField>

<ParamField query="TotalCredit" type="RangeSearchFilter<decimal>">
  Filtra por el total del crédito del reembolso.
</ParamField>

<ParamField query="TotalAmount" type="RangeSearchFilter<decimal>">
  Filtra por el monto total del reembolso.
</ParamField>

<ParamField query="InsertedBy" type="StringSearchFilter">
  Filtra por el usuario que creó el reembolso.
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request GET \
  --url "https://api.primenetcloud.com/v1/reimbursement-liabilities?start=2026-07-01T00:00:00Z&end=2026-07-31T23:59:59Z&skip=20&take=20" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID"
```

```javascript JavaScript
const response = await fetch(
  "https://api.primenetcloud.com/v1/reimbursement-liabilities?start=2026-07-01T00:00:00Z&end=2026-07-31T23:59:59Z&skip=20&take=20",
  {
    method: "GET",
    headers: {
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "TENANT_ID"
    },
  }
);

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.primenetcloud.com/v1/reimbursement-liabilities",
    params={
        "start": "2026-07-01T00:00:00Z",
        "end": "2026-07-31T23:59:59Z",
        "skip": 20,
        "take": 20,
    },
   headers={
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "TENANT_ID"
    }
)

print(response.json())
```
