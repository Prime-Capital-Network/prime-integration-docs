---
title: "Eliminar Detalle"
---

Elimina un detalle específico asociado a un reembolso.

```http
DELETE /reimbursement-liabilities/{id}/details/{detailId}
```

<Note>
  Este endpoint requiere el scope **`ReimbursementLiabilities.Write`**.
</Note>

### Path parameters

<ParamField path="id" type="integer" required>
  Identificador único del reembolso al que pertenece el detalle.
</ParamField>

<ParamField path="detailId" type="integer" required>
  Identificador único del detalle a eliminar.
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request DELETE \
  --url "https://api.primenetcloud.com/v1/reimbursement-liabilities/1054/details/1" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID"
```

```javascript JavaScript
const response = await fetch("https://api.primenetcloud.com/v1/reimbursement-liabilities/1054/details/1", {
  method: "DELETE",
  headers: {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Tenant-Id": "TENANT_ID",
  },
});

console.log(response.status); // 204
```

```python Python
import requests

response = requests.delete(
    "https://api.primenetcloud.com/v1/reimbursement-liabilities/1054/details/1",
    headers={
        "X-Api-Key": "YOUR_API_KEY",
        "X-Tenant-Id": "TENANT_ID",
    },
)

print(response.status_code) # 204
```

</CodeGroup>

### Response

Si la eliminación es exitosa, el endpoint retorna `204 No Content` sin body.

### Errores

<ResponseField name="400 Bad Request">
  La solicitud será rechazada si el reembolso ya ha sido posteada (campo `status` como `Posted`). el reembolso no puede ser modificada en este estado.
</ResponseField>

<ResponseField name="403 Forbidden">
  El API Key no tiene el scope `ReimbursementLiabilities.Write` requerido.
</ResponseField>

<ResponseField name="404 Not Found">
  No existe ningun reembolso con el `id` especificado, o el `detailId` no pertenece a el reembolso.
</ResponseField>
