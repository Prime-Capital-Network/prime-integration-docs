title: "Obtener reembolso Por ID"
sidebarTitle: "Obtener por ID"
description: "Permite extraer un reembolso (cuenta por pagar) en específico mediante REST API"
---

Obtén un reembolso por ID en la empresa.

```http
GET /reimbursement-liabilities/{id}
```

<Note>
  Este endpoint requiere el scope **`ReimbursementLiabilities.Read`**.
</Note>

### Path parameters

<ParamField path="id" type="integer" required>
  Identificador único del reembolso.
</ParamField>

### Query parameters

<ParamField query="include" type="string[]">
  Secciones adicionales a incluir en la respuesta. Valores aceptados:

  - `Details`
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request GET \
  --url "https://api.primenetcloud.com/v1/reimbursement-liabilities/1234?include=Details" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID"
```

```javascript JavaScript
const response = await fetch(
  "https://api.primenetcloud.com/v1/reimbursement-liabilities/1234?include=Details",
  {
    method: "GET",
    headers: {
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "e3d5e6af-3ec2-49c1-92bd-9632171e9a66"
    },
  }
);
const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.primenetcloud.com/v1/reimbursement-liabilities/1234",
    params={"include": ["Details"]},
    headers={
      "X-Api-Key": "pnk_abc123.s3cr3tValue",
      "X-Tenant-Id": "e3d5e6af-3ec2-49c1-92bd-9632171e9a66"
    }
)
print(response.json())
```

</CodeGroup>

</ResponseField>

<ResponseField name="date" type="date">

</ResponseField>

</ResponseField>
