---
title: "Eliminar Adjunto"
---

Elimina un archivo adjunto específico.

<Note>
  Este endpoint requiere el scope **`Attachments.Write`**.
</Note>

### Path parameters

<ParamField path="id" type="integer" required>
  Identificador único del archivo adjunto que se desea eliminar.
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request DELETE \
  --url "https://api.primenetcloud.com/v1/attachments/2045" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID"
```

```javascript JavaScript
const response = await fetch("[https://api.primenetcloud.com/v1/attachments/2045", {
  method: "DELETE",
  headers: {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Tenant-Id": "TENANT_ID",
  },
});

console.log(response.status); // 204
```

```python Python
import requests

response = requests.delete(
    "https://api.primenetcloud.com/v1/attachments/2045",
    headers={
        "X-Api-Key": "YOUR_API_KEY",
        "X-Tenant-Id": "TENANT_ID",
    },
)

print(response.status_code) 
```

</CodeGroup>

### Response

Si la eliminación es exitosa, el endpoint retorna `204 No Content` sin body.

### Errores

<ResponseField name="403 Forbidden">
  El API Key no tiene el scope `Attachments.Write` requerido.
</ResponseField>

<ResponseField name="404 Not Found">
  No existe ningún archivo adjunto con el `id` especificado.
</ResponseField>
