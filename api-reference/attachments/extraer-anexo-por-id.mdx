---
title: "Obtener Adjunto por ID"
sidebarTitle: "Obtener por ID"
---

Obtén un adjunto por ID en la empresa.

```http
GET /attachments/{id}
```

<Note>
  Este endpoint requiere el scope **`Attachments.Read`**.
</Note>

### Path parameters

<ParamField path="id" type="integer" required>
  Identificador único del adjunto.
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request GET \
  --url "https://api.primenetcloud.com/v1/attachments/501" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID"
```

```javascript JavaScript
const response = await fetch("https://api.primenetcloud.com/v1/attachments/501", {
  method: "GET",
  headers: {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Tenant-Id": "TENANT_ID"
  },
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.primenetcloud.com/v1/attachments/501",
    headers={
        "X-Api-Key": "YOUR_API_KEY",
        "X-Tenant-Id": "TENANT_ID"
    }
)

print(response.json())
```

</CodeGroup>

### Response fields

<ResponseField name="id" type="integer">
  Identificador único del adjunto.
</ResponseField>

<ResponseField name="name" type="string">
  Nombre del adjunto.
</ResponseField>

<ResponseField name="description" type="string">
  Descripción del adjunto.
</ResponseField>

<ResponseField name="category" type="object">
  Categoría del adjunto.

  <Expandable title="propiedades">
    <ResponseField name="id" type="integer">
      Identificador de la categoría.
    </ResponseField>

    <ResponseField name="name" type="string">
      Nombre de la categoría.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="fileName" type="string">
  Nombre del archivo, sin extensión.
</ResponseField>

<ResponseField name="fileExtension" type="string">
  Extensión del archivo.
</ResponseField>

<ResponseField name="contentType" type="string">
  Tipo de contenido (MIME type) del archivo.
</ResponseField>

<ResponseField name="menuId" type="string">
  Identificador del menú (módulo) al cual pertenece el registro asociado.
</ResponseField>

<ResponseField name="length" type="integer">
  Tamaño del archivo en bytes.
</ResponseField>

<ResponseField name="recordId" type="string">
  Identificador del registro al cual está asociado el adjunto.
</ResponseField>

<ResponseField name="trackingId" type="integer">
  Identificador de seguimiento (tracking) asociado al adjunto.
</ResponseField>

<ResponseField name="insertedBy" type="string">
  Usuario que creó el adjunto.
</ResponseField>

<ResponseField name="insertedDate" type="string">
  Fecha y hora de creación del adjunto, en formato ISO 8601.
</ResponseField>

<ResponseField name="isActive" type="boolean">
  Indica si el adjunto está activo.
</ResponseField>

### Errores

<ResponseField name="404 Not Found">
  No existe ningún adjunto con el `id` especificado en la URL.
</ResponseField>

<ResponseField name="403 Forbidden">
  El API Key no tiene el scope `Attachments.Read` requerido.
</ResponseField>
