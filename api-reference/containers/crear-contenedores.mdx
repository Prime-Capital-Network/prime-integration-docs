---
title: "Crear"
---

Crea un nuevo container en la empresa.

```http
POST /containers
```

<Note>
  Este endpoint requiere el scope **`Containers.Write`**.
</Note>

### Body parameters

<ParamField body="typeId" type="string" required>
  Identificador del tipo de contenedor.
</ParamField>

<ParamField body="number" type="string">
  Número del contenedor. Máximo 13 caracteres.
</ParamField>

<ParamField body="seals" type="string[]">
  Sellos asociados al contenedor. Máximo 4 valores. Máximo 20 caracteres por valor.
</ParamField>

<ParamField body="marks" type="string[]">
  Marcas asociadas al contenedor. Máximo 3 registros. Máximo 50 caracteres por valor.
</ParamField>

<ParamField body="vendorId" type="integer">
  Identificador del vendor asociado.

  - Debe ser un asociado con el campo `IsVendor` como true.
</ParamField>

<ParamField body="loadedDate" type="date">
  Fecha de carga del contenedor.
</ParamField>

<ParamField body="arrivalDate" type="date">
  Fecha de llegada del contenedor.
</ParamField>

<ParamField body="returnedDate" type="date">
  Fecha de devolución del contenedor.

  - Debe ser mayor o igual a `arrivalDate` si ambos campos están presentes.
</ParamField>

<ParamField body="freeDays" type="integer">
  Cantidad de días libres asignados. Debe ser mayor o igual a cero.
</ParamField>

<ParamField body="tariff" type="decimal">
  Tarifa asociada al contenedor. Debe ser mayor o igual a cero.
</ParamField>

<ParamField body="masterId" type="integer">
  Identificador del master asociado.

  - El campo `TransportType` del master no puede ser `Air`.
</ParamField>

<ParamField body="weightValue" type="decimal">
  Peso del contenedor. Requiere `weightUnit` si se define. Debe ser mayor o igual a cero.
</ParamField>

<ParamField body="weightUnit" type="string">
  Unidad de peso. Requerido si `weightValue` no es null. Valores aceptados:

  - `Kilograms`
  - `Tons`
  - `Pounds`
</ParamField>

<ParamField body="volumeValue" type="decimal">
  Volumen del contenedor. Requiere `volumeUnit` si se define. Debe ser mayor o igual a cero.
</ParamField>

<ParamField body="volumeUnit" type="string">
  Unidad de volumen. Requerido si `volumeValue` no es null. Valores aceptados:

  - `CubicMeters`
  - `CubicFeet`
</ParamField>

<ParamField body="pieces" type="integer">
  Cantidad de piezas dentro del contenedor. Debe ser mayor o igual a cero.
</ParamField>

<ParamField body="packingTypeId" type="integer">
  Identificador del tipo de empaque.
</ParamField>

<ParamField body="customId" type="integer">
  Identificador del registro de aduana (customs) asociado.
</ParamField>

<ParamField body="groundTransportationId" type="integer">
  Identificador del transporte terrestre asociado.
</ParamField>

<ParamField body="description" type="string">
  Descripción del contenedor. Máximo 250 caracteres.
</ParamField>

<ParamField body="length" type="integer">
  Largo del contenedor. Debe ser mayor o igual a cero.
</ParamField>

<ParamField body="width" type="integer">
  Ancho del contenedor. Debe ser mayor o igual a cero.
</ParamField>

<ParamField body="height" type="integer">
  Alto del contenedor. Debe ser mayor o igual a cero.
</ParamField>

<ParamField body="dimensionUnit" type="string">
  Unidad de las dimensiones (`length`, `width`, `height`). Valores aceptados:

  - `Meters`
  - `Inches`
  - `Centimeters`
  - `Milimeters`
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request POST \
  --url "https://api.primenetcloud.com/v1/containers" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID" \
  --header "Content-Type: application/json" \
  --data '{
    "typeId": "12TR",
    "number": "MSCU1234567",
    "vendorId": 12,
    "loadedDate": "2026-07-01",
    "arrivalDate": "2026-07-11",
    "freeDays": 5,
    "weightValue": 18500,
    "weightUnit": "Kilograms",
    "pieces": 320
  }'
```

```javascript JavaScript
const response = await fetch("https://api.primenetcloud.com/v1/containers", {
  method: "POST",
  headers: {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Tenant-Id": "TENANT_ID",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    typeId: "12TR",
    number: "MSCU1234567",
    vendorId: 12,
    loadedDate: "2026-07-01",
    arrivalDate: "2026-07-11",
    freeDays: 5,
    weightValue: 18500,
    weightUnit: "Kilograms",
    pieces: 320,
  }),
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    "https://api.primenetcloud.com/v1/containers",
    json={
        "typeId": "12TR",
        "number": "MSCU1234567",
        "vendorId": 12,
        "loadedDate": "2026-07-01",
        "arrivalDate": "2026-07-11",
        "freeDays": 5,
        "weightValue": 18500,
        "weightUnit": "Kilograms",
        "pieces": 320,
    },
    headers={
        "X-Api-Key": "YOUR_API_KEY",
        "X-Tenant-Id": "TENANT_ID",
    },
)

print(response.json())
```

</CodeGroup>

### Example response

```json
{
  "id": 501
}
```

### Response fields

<ResponseField name="id" type="integer">
  Identificador único del contenedor creado.
</ResponseField>

### Errores

<ResponseField name="403 Forbidden" type="error">
  El API Key no tiene el scope `Containers.Write` requerido.
</ResponseField>
