---
title: "Listar"
---

Obtén una lista paginada de containers en tu empresa.

```http
GET /containers
```

<Note>
  Este endpoint requiere el scope **`Containers.Read`**.
</Note>

### Query parameters

<ParamField query="master_id" type="integer">
  Filtra los containers asociados a un master específico.
</ParamField>

<ParamField query="sort_by" type="string">
  Campo por el cual ordenar los resultados. Valores aceptados:

  - `Id`
  - `Type.Id`
  - `Type.Name`
  - `Number`
  - `Vendor.Name`
  - `LoadedDate`
  - `ArrivalDate`
  - `ReturnedDate`
  - `FreeDays`
  - `Tariff`
  - `PackingType.Name`
  - `Pieces`
</ParamField>

<ParamField query="skip" default="0" type="integer">
  Número de registros a omitir desde el inicio de los resultados. Utilízado junto con `take` para paginar los resultados.
</ParamField>

<ParamField query="take" default="20" type="integer">
  Número de registros a retornar. Por defecto `20`, máximo `100`.
</ParamField>

<ParamField query="Type.Id" type="StringSearchFilter">
  Filtra por el identificador del tipo de contenedor.
</ParamField>

<ParamField query="Type.Name" type="StringSearchFilter">
  Filtra por el nombre del tipo de contenedor.
</ParamField>

<ParamField query="Number" type="StringSearchFilter">
  Filtra por el número del contenedor.
</ParamField>

<ParamField query="Seal" type="StringSearchFilter">
  Filtra por un sello (seal) asociado al contenedor.
</ParamField>

<ParamField query="LoadedDate" type="RangeSearchFilter<date>">
  Filtra por la fecha de carga del contenedor.
</ParamField>

<ParamField query="ArrivalDate" type="RangeSearchFilter<date>">
  Filtra por la fecha de llegada del contenedor.
</ParamField>

<ParamField query="ReturnedDate" type="RangeSearchFilter<date>">
  Filtra por la fecha de devolución del contenedor.
</ParamField>

<ParamField query="Vendor.Name" type="StringSearchFilter">
  Filtra por el nombre del vendor asociado.
</ParamField>

<ParamField query="FreeDays" type="RangeSearchFilter<integer>">
  Filtra por la cantidad de días libres asignados.
</ParamField>

<ParamField query="Tariff" type="RangeSearchFilter<decimal>">
  Filtra por la tarifa asociada al contenedor.
</ParamField>

<ParamField query="Pieces" type="RangeSearchFilter<integer>">
  Filtra por la cantidad de piezas dentro del contenedor.
</ParamField>

<ParamField query="PackingType.Name" type="StringSearchFilter">
  Filtra por el nombre del tipo de empaque.
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request GET \
  --url "https://api.primenetcloud.com/v1/containers?master_id=45&skip=20&take=20" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID"
```

```javascript JavaScript
const response = await fetch(
  "https://api.primenetcloud.com/v1/containers?master_id=45&skip=20&take=20",
  {
    method: "GET",
    headers: {
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "TENANT_ID"
    },
  }
);

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.primenetcloud.com/v1/containers",
    params={
        "master_id": 45,
        "skip": 20,
        "take": 20,
    },
   headers={
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "TENANT_ID"
    }
)

print(response.json())
```

</CodeGroup>
