---
title: "Listar"
---

Obtén una lista paginada de despachos aduanales en tu empresa.

```http
GET /customs
```

<Note>
  Este endpoint requiere el scope **`Customs.Read`**.
</Note>

### Query parameters

<ParamField query="sort_by" type="string">
  Campo por el cual ordenar los resultados. Valores aceptados:

  - `TransactionNumber`
  - `House.Id`
  - `House.ETD`
  - `House.ETA`
  - `SalesPerson.Name`
  - `Regime.Name`
  - `CustomStatus.Id`
  - `CustomStatus.Name`
  - `Dispatcher.Name`
  - `AssignedTo.FullName`
  - `DeclarationDate`
  - `IsCargoHeld`
  - `Status`
  - `InsertedDate`
</ParamField>

<ParamField query="skip" default="0" type="integer">
  Número de registros a omitir desde el inicio de los resultados. Utilízado junto con `take` para paginar los resultados.
</ParamField>

<ParamField query="take" default="20" type="integer">
  Número de registros a retornar. Por defecto `20`, máximo `100`.
</ParamField>

<ParamField query="TransactionNumber" type="RangeSearchFilter<integer>">
  Filtra por número de transacción del despacho.
</ParamField>

<ParamField query="TrackingNumber" type="RangeSearchFilter<integer>">
  Filtra por número de tracking del despacho.
</ParamField>

<ParamField query="House.ETD" type="RangeSearchFilter<datetime>">
  Filtra por la fecha estimada de salida del house asociado.
</ParamField>

<ParamField query="House.ETA" type="RangeSearchFilter<datetime>">
  Filtra por la fecha estimada de llegada del house asociado.
</ParamField>

<ParamField query="House.BL" type="StringSearchFilter">
  Filtra por el número de BL del house asociado.
</ParamField>

<ParamField query="SalesPerson.Name" type="StringSearchFilter">
  Filtra por el nombre del vendedor asignado.
</ParamField>

<ParamField query="Regime.Name" type="StringSearchFilter">
  Filtra por el nombre del régimen aduanero.
</ParamField>

<ParamField query="CustomStatus.Name" type="StringSearchFilter">
  Filtra por el nombre del estatus aduanal.
</ParamField>

<ParamField query="Dispatcher.Name" type="StringSearchFilter">
  Filtra por el nombre del despachante.
</ParamField>

<ParamField query="BillTo.Name" type="StringSearchFilter">
  Filtra por el nombre del asociado a quien se factura.
</ParamField>

<ParamField query="AssignedTo.FullName" type="StringSearchFilter">
  Filtra por el nombre completo del usuario asignado.
</ParamField>

<ParamField query="PortOfLoading.Name" type="StringSearchFilter">
  Filtra por el nombre del puerto de carga del house asociado.
</ParamField>

<ParamField query="PortOfDischarge.Name" type="StringSearchFilter">
  Filtra por el nombre del puerto de descarga del house asociado.
</ParamField>

<ParamField query="DeclarationDate" type="RangeSearchFilter<date>">
  Filtra por la fecha de declaración.
</ParamField>

<ParamField query="IsCargoHeld" type="BooleanSearchFilter">
  Filtra despachos cuya carga está retenida.
</ParamField>

<ParamField query="TransportType.Id" type="EnumSearchFilter">
  Filtra por el tipo de transporte del house asociado. Valores aceptados:

  - `Air`
  - `Sea`
  - `Ground`
  - `Rail`
</ParamField>

<ParamField query="DeclarationStatus.Id" type="EnumSearchFilter">
  Filtra por el estatus de la declaración. Valores aceptados:

  - `Expired`
  - `ToExpire`
  - `InProgress`
</ParamField>

<ParamField query="Status" type="EnumSearchFilter">
  Filtra por el estatus del proceso aduanal. Valores aceptados:

  - `Cancelled`
  - `Quoted`
  - `Unassigned`
  - `InCoordination`
  - `AtCustoms`
  - `AvailableForDelivery`
  - `Completed`
  - `Declined`
</ParamField>

<ParamField query="InsertedDate" type="RangeSearchFilter<datetime>">
  Filtra por la fecha de creación del despacho.
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request GET \
  --url "https://api.primenetcloud.com/v1/customs?IsCargoHeld=true&skip=20&take=20" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID"
```

```javascript JavaScript
const response = await fetch(
  "https://api.primenetcloud.com/v1/customs?IsCargoHeld=true&skip=20&take=20",
  {
    method: "GET",
    headers: {
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "TENANT_ID"
    },
  }
);

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.primenetcloud.com/v1/customs",
    params={
        "IsCargoHeld": "true",
        "skip": 20,
        "take": 20,
    },
   headers={
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "TENANT_ID"
    }
)

print(response.json())
```

</CodeGroup>
