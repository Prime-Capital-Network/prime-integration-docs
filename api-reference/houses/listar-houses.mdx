Obtén una lista paginada de houses en la empresa.

```http
GET /houses
```

<Note>
  Este endpoint requiere el scope **`Houses.Read`**.
</Note>

### Query parameters

<ParamField query="sort_by" type="string">
  Campo por el cual ordenar los resultados. Valores aceptados:

  - `Id`
  - `TransactionNumber`
  - `TrackingNumber`
  - `MovementType`
  - `TransportType`
  - `SalesPerson.Name`
  - `BL`
  - `Shipper.Name`
  - `Consignee.Name`
  - `OriginAgent.Name`
  - `DestinationAgent.Name`
  - `PortOfLoading.Id`
  - `PortOfDischarge.Id`
  - `ETD`
  - `ETA`
  - `InsertedDate`
  - `Status`
</ParamField>

<ParamField query="skip" default="0" type="integer">
  Número de registros a omitir desde el inicio de los resultados. Utilízado junto con `take` para paginar los resultados.
</ParamField>

<ParamField query="take" default="20" type="integer">
  Número de registros a retornar. Por defecto `20`, máximo `100`.
</ParamField>

<ParamField query="TransactionNumber" type="RangeSearchFilter<integer>">
  Filtra por número de transacción del house.
</ParamField>

<ParamField query="TrackingNumber" type="RangeSearchFilter<integer>">
  Filtra por número de tracking del house.
</ParamField>

<ParamField query="Master.ATD" type="RangeSearchFilter<datetime>">
  Filtra por la fecha real de salida (ATD) del master.
</ParamField>

<ParamField query="Master.ATA" type="RangeSearchFilter<datetime>">
  Filtra por la fecha real de llegada (ATA) del master.
</ParamField>

<ParamField query="Master.MasterBL" type="StringSearchFilter">
  Filtra por el número de BL del master.
</ParamField>

<ParamField query="Master.TransactionNumber" type="RangeSearchFilter<integer>">
  Filtra por el número de transacción del master.
</ParamField>

<ParamField query="MovementType.Id" type="EnumSearchFilter">
  Filtra por el tipo de movimiento del house.
</ParamField>

<ParamField query="TransportType.Id" type="EnumSearchFilter">
  Filtra por el tipo de transporte del house.
</ParamField>

<ParamField query="ContainerType.Id" type="EnumSearchFilter">
  Filtra por el tipo de contenedor del house.
</ParamField>

<ParamField query="Customer.Id" type="RangeSearchFilter<integer>">
  Filtra por el identificador del cliente.
</ParamField>

<ParamField query="Customer.Name" type="StringSearchFilter">
  Filtra por el nombre del cliente.
</ParamField>

<ParamField query="IsBilled" type="BooleanSearchFilter">
  Filtra houses que ya han sido facturados.
</ParamField>

<ParamField query="SalesPerson.Name" type="StringSearchFilter">
  Filtra por el nombre del vendedor asignado.
</ParamField>

<ParamField query="BL" type="StringSearchFilter">
  Filtra por el número de BL del house.
</ParamField>

<ParamField query="Shipper.Type" type="string">
  Filtra por el tipo de contacto del shipper. Valores aceptados:

  - `Associate`
  - `CustomerAssociate`
</ParamField>

<ParamField query="Shipper.Name" type="StringSearchFilter">
  Filtra por el nombre del shipper.
</ParamField>

<ParamField query="Consignee.Type" type="string">
  Filtra por el tipo de contacto del consignee. Valores aceptados:

  - `Associate`
  - `CustomerAssociate`
</ParamField>

<ParamField query="Consignee.Name" type="StringSearchFilter">
  Filtra por el nombre del consignee.
</ParamField>

<ParamField query="Agent.Name" type="StringSearchFilter">
  Filtra por el nombre del agente. Aplica sobre el agente de origen o de destino según el tipo de movimiento.
</ParamField>

<ParamField query="PortOfLoading.Id" type="StringSearchFilter">
  Filtra por el identificador del puerto de carga.
</ParamField>

<ParamField query="PortOfDischarge.Id" type="StringSearchFilter">
  Filtra por el identificador del puerto de descarga.
</ParamField>

<ParamField query="ETD" type="RangeSearchFilter<datetime>">
  Filtra por la fecha estimada de salida.
</ParamField>

<ParamField query="ETA" type="RangeSearchFilter<datetime>">
  Filtra por la fecha estimada de llegada.
</ParamField>

<ParamField query="Kilograms" type="RangeSearchFilter<decimal>">
  Filtra por el peso en kilogramos.
</ParamField>

<ParamField query="CubicMeters" type="RangeSearchFilter<decimal>">
  Filtra por el volumen en metros cúbicos.
</ParamField>

<ParamField query="AssignedTo.Id" type="StringSearchFilter">
  Filtra por el identificador del usuario asignado.
</ParamField>

<ParamField query="InsertedDate" type="RangeSearchFilter<datetime>">
  Filtra por la fecha de creación del house.
</ParamField>

<ParamField query="Status.Id" type="EnumSearchFilter">
  Filtra por el estatus del house.
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request GET \
  --url "https://api.primeerp.com/v1/houses?Status.Id=InCoordination&skip=20&take=20" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID"
```

```javascript JavaScript
const response = await fetch(
  "https://api.primeerp.com/v1/houses?Status.Id=InCoordination&skip=20&take=20",
  {
    method: "GET",
    headers: {
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "TENANT_ID"
    },
  }
);

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.primeerp.com/v1/houses",
    params={
        "Status.Id": "InCoordination",
        "skip": 20,
        "take": 20,
    },
   headers={
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "TENANT_ID"
    }
)

print(response.json())
```

</CodeGroup>
