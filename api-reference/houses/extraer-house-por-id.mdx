---

Obtén un house por ID en la empresa.

```http
GET /houses/{id}
```

<Note>
  Este endpoint requiere el scope **`Houses.Read`**.
</Note>

### Path parameters

<ParamField path="id" type="integer" required>
  Identificador único del house.
</ParamField>

### Query parameters

<ParamField query="include" type="string">
  Incluye los detalles a traer en el mismo request. Valores aceptados: `Packings`, `Vehicles`, `Routes`, `Events`, y `Containers`. De no ser enviados, el API devuelve un array vacío para cada uno.
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request GET \
  --url "https://api.primenetcloud.com/v1/houses/5321?include=Vehicles" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID"
```

```javascript JavaScript
const response = await fetch(
  "https://api.primenetcloud.com/v1/houses/5321?include=Vehicles",
  {
    method: "GET",
    headers: {
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "e3d5e6af-3ec2-49c1-92bd-9632171e9a66"
    },
  }
);
const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.primenetcloud.com/v1/houses/5321",
    params={"include": "Vehicles"},
    headers={
      "X-Api-Key": "pnk_abc123.s3cr3tValue",
      "X-Tenant-Id": "e3d5e6af-3ec2-49c1-92bd-9632171e9a66"
    }
)
print(response.json())
```

</CodeGroup>
