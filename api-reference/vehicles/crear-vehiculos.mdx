---
title: "Crear"
---

Crea un nuevo vehículo en la empresa.

```http
POST /vehicles
```

<Note>
  Este endpoint requiere el scope **`Vehicles.Write`**.
</Note>

### Body parameters

<ParamField body="description" type="string" required>
  Descripción del vehículo. Máximo 80 caracteres.
</ParamField>

<ParamField body="vehicleTypeId" type="integer" required>
  Identificador del tipo de vehículo.
</ParamField>

<ParamField body="brandId" type="integer">
  Identificador de la marca del vehículo.
</ParamField>

<ParamField body="modelId" type="integer">
  Identificador del modelo del vehículo.
</ParamField>

<ParamField body="year" type="integer">
  Año del vehículo.

  - Debe ser mayor o igual a 1886.
  - Debe ser menor o igual al próximo año.
</ParamField>

<ParamField body="color" type="string">
  Color del vehículo. Máximo 15 caracteres.
</ParamField>

<ParamField body="plate" type="string">
  Placa del vehículo. Máximo 12 caracteres.
</ParamField>

<ParamField body="vin" type="string">
  Número de identificación vehicular (VIN). Máximo 20 caracteres.
</ParamField>

<ParamField body="motor" type="string">
  Número de motor. Máximo 20 caracteres.
</ParamField>

<ParamField body="tons" type="integer">
  Capacidad de carga en toneladas.
</ParamField>

<ParamField body="passengers" type="integer">
  Capacidad de pasajeros.
</ParamField>

<ParamField body="capacityValue" type="integer">
  Valor de la capacidad del vehículo.
</ParamField>

<ParamField body="capacityUnit" type="string">
  Unidad de medida de la capacidad (peso). Valores aceptados:

  - `Kilograms`
  - `Tons`
  - `Pounds`
</ParamField>

<ParamField body="acquiredMileage" type="integer">
  Millaje/kilometraje con el que fue adquirido el vehículo.
</ParamField>

<ParamField body="actualMileage" type="integer">
  Millaje/kilometraje actual del vehículo.
</ParamField>

<ParamField body="mileageUnit" type="string">
  Unidad de medida del millaje. Valores aceptados:

  - `Miles`
  - `Kilometers`
</ParamField>

<ParamField body="fuelTypeId" type="integer">
  Identificador del tipo de combustible.
</ParamField>

<ParamField body="fuelCapacity" type="integer">
  Capacidad del tanque de combustible.
</ParamField>

<ParamField body="fuelCapacityUnit" type="string">
  Unidad de medida de la capacidad de combustible. Valores aceptados:

  - `Gallon`
  - `Liter`
</ParamField>

<ParamField body="horsePower" type="integer">
  Potencia del motor en caballos de fuerza (HP).
</ParamField>

<ParamField body="vehicleOwnership" type="string">
  Tipo de propiedad del vehículo. Valores aceptados:

  - `Owned`
  - `Leased`
  - `Rented`
</ParamField>

<ParamField body="vendorId" type="integer">
  Identificador del vendor (proveedor) asociado, aplica si el vehículo es alquilado/leaseado.
</ParamField>

<ParamField body="driverId" type="integer">
  Identificador del conductor asignado al vehículo.
</ParamField>

<ParamField body="insuranceId" type="integer">
  Identificador de la aseguradora.
</ParamField>

<ParamField body="insuraceExpiration" type="date">
  Fecha de vencimiento del seguro, en formato `YYYY-MM-DD`.
</ParamField>

<ParamField body="policy" type="string">
  Número de póliza de seguro. Máximo 20 caracteres.
</ParamField>

<ParamField body="front" type="decimal">
  Ancho del vehículo. Debe ser mayor que cero. Precisión de hasta 2 decimales.
</ParamField>

<ParamField body="length" type="decimal">
  Largo del vehículo. Debe ser mayor que cero. Precisión de hasta 2 decimales.
</ParamField>

<ParamField body="height" type="decimal">
  Alto del vehículo. Debe ser mayor que cero. Precisión de hasta 2 decimales.
</ParamField>

<ParamField body="dimensionUnit" type="string">
  Unidad de medida de las dimensiones del vehículo. Valores aceptados:

  - `CubicMeters`
  - `CubicFeet`
</ParamField>

<ParamField body="cargoFront" type="decimal">
  Ancho del área de carga. Debe ser mayor que cero. Precisión de hasta 2 decimales.
</ParamField>

<ParamField body="cargoLength" type="decimal">
  Largo del área de carga. Debe ser mayor que cero. Precisión de hasta 2 decimales.
</ParamField>

<ParamField body="cargoHeight" type="decimal">
  Alto del área de carga. Debe ser mayor que cero. Precisión de hasta 2 decimales.
</ParamField>

<ParamField body="cargoDimensionUnit" type="string">
  Unidad de medida de las dimensiones del área de carga. Valores aceptados:

  - `CubicMeters`
  - `CubicFeet`
</ParamField>

<ParamField body="doors" type="integer">
  Cantidad de puertas del vehículo.
</ParamField>

<ParamField body="combinedCapacity" type="integer">
  Capacidad combinada del vehículo. Debe ser mayor o igual a cero.
</ParamField>

<ParamField body="emptyWeight" type="integer">
  Peso vacío del vehículo. Debe ser mayor o igual a cero.
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request POST \
  --url "https://api.primenetcloud.com/v1/vehicles" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID" \
  --header "Content-Type: application/json" \
  --data '{
    "description": "Camión de reparto 01",
    "vehicleTypeId": 3,
    "brandId": 12,
    "modelId": 45,
    "year": 2022,
    "plate": "A123456",
    "vin": "1HGCM82633A004352",
    "mileageUnit": "Kilometers",
    "vehicleOwnership": "Owned"
  }'
```

```javascript JavaScript
const response = await fetch("https://api.primenetcloud.com/v1/vehicles", {
  method: "POST",
  headers: {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Tenant-Id": "TENANT_ID",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    description: "Camión de reparto 01",
    vehicleTypeId: 3,
    brandId: 12,
    modelId: 45,
    year: 2022,
    plate: "A123456",
    vin: "1HGCM82633A004352",
    mileageUnit: "Kilometers",
    vehicleOwnership: "Owned"
  }),
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    "https://api.primenetcloud.com/v1/vehicles",
    json={
        "description": "Camión de reparto 01",
        "vehicleTypeId": 3,
        "brandId": 12,
        "modelId": 45,
        "year": 2022,
        "plate": "A123456",
        "vin": "1HGCM82633A004352",
        "mileageUnit": "Kilometers",
        "vehicleOwnership": "Owned"
    },
    headers={
        "X-Api-Key": "YOUR_API_KEY",
        "X-Tenant-Id": "TENANT_ID",
    },
)

print(response.json())
```

</CodeGroup>

### Example response

```json
{
  "id": 87
}
```

### Response fields

<ResponseField name="id" type="integer">
  Identificador único del vehículo creado.
</ResponseField>

### Errores

<ResponseField name="403 Forbidden">
  El API Key no tiene el scope `Vehicles.Write` requerido.
</ResponseField>
