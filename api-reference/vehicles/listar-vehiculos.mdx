---
title: "Listar"
---

Obtén una lista paginada de vehículos en la empresa.

```http
GET /vehicles
```

<Note>
  Este endpoint requiere el scope **`Vehicles.Read`**.
</Note>

### Query parameters

<ParamField query="sort_by" type="string">
  Campo por el cual ordenar los resultados. Valores aceptados:

  - `Id`
  - `Description`
  - `Year`
  - `VIN`
  - `Plate`
  - `ActualMileage`
  - `Driver.Id`
  - `Driver.Name`
  - `Brand.Name`
  - `Model.Name`
  - `VehicleType.Name`
  - `Vendor.Id`
  - `Vendor.Name`
  - `InsertedBy`
</ParamField>

<ParamField query="skip" default="20" type="integer">
  Número de registros a omitir desde el inicio de los resultados. Utilízado junto con `take` para paginar los resultados.
</ParamField>

<ParamField query="take" type="integer">
  Número de registros a retornar. Por defecto `20`, máximo `100`.
</ParamField>

<ParamField query="Description" type="StringSearchFilter">
  Filtra por la descripción del vehículo.
</ParamField>

<ParamField query="Driver.Name" type="StringSearchFilter">
  Filtra por el nombre del conductor asignado.
</ParamField>

<ParamField query="Year" type="RangeSearchFilter<integer>">
  Filtra por el año del vehículo.
</ParamField>

<ParamField query="Brand.Name" type="StringSearchFilter">
  Filtra por el nombre de la marca.
</ParamField>

<ParamField query="Model.Name" type="StringSearchFilter">
  Filtra por el nombre del modelo.
</ParamField>

<ParamField query="VehicleType.Name" type="StringSearchFilter">
  Filtra por el nombre del tipo de vehículo.
</ParamField>

<ParamField query="VIN" type="StringSearchFilter">
  Filtra por el número de identificación vehicular (VIN).
</ParamField>

<ParamField query="Plate" type="StringSearchFilter">
  Filtra por la placa del vehículo.
</ParamField>

<ParamField query="ActualMileage" type="RangeSearchFilter<integer>">
  Filtra por el kilometraje/millaje actual del vehículo.
</ParamField>

<ParamField query="MileageUnit.Id" type="EnumSearchFilter">
  Filtra por la unidad de medida del millaje.
  
  Valores aceptados:

  - `Miles`
  - `Kilometers`
</ParamField>

<ParamField query="VehicleOwnership.Id" type="EnumSearchFilter">
  Filtra por el tipo de propiedad del vehículo.
  
  Valores aceptados:

  - `Owned`
  - `Affiliate`
</ParamField>

<ParamField query="Vendor.Name" type="StringSearchFilter">
  Filtra por el nombre del vendor (proveedor) asociado.
</ParamField>

<ParamField query="Status.Id" type="EnumSearchFilter">
  Filtra por el estatus del vehículo.
  
  Valores aceptados:

  - `Active`
  - `UnderMaintenance`
  - `UnderRepair`
  - `OutOfService`
  - `Retired`
  - `Deleted`
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request GET \
  --url "https://api.primenetcloud.com/v1/vehicles?skip=0&take=20" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID"
```

```javascript JavaScript
const response = await fetch(
  "https://api.primenetcloud.com/v1/vehicles?skip=0&take=20",
  {
    method: "GET",
    headers: {
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "TENANT_ID"
    },
  }
);

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.primenetcloud.com/v1/vehicles",
    params={
        "skip": 0,
        "take": 20,
    },
    headers={
        "X-Api-Key": "YOUR_API_KEY",
        "X-Tenant-Id": "TENANT_ID"
    }
)

print(response.json())
```

</CodeGroup>
