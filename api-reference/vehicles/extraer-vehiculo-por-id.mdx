---
title: "Obtener Vehículo Por ID"
sidebarTitle: "Obtener por ID"
description: "Permite extraer un vehículo en específico mediante REST API"
---

Obtén un vehículo por ID en la empresa.

```http
GET /vehicles/{id}
```

<Note>
  Este endpoint requiere el scope **`Vehicles.Read`**.
</Note>

### Path parameters

<ParamField path="id" type="integer" required>
  Identificador único del vehículo.
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request GET \
  --url "https://api.primenetcloud.com/v1/vehicles/87" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID"
```

```javascript JavaScript
const response = await fetch("https://api.primenetcloud.com/v1/vehicles/87", {
  method: "GET",
  headers: {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Tenant-Id": "TENANT_ID"
  },
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.primenetcloud.com/v1/vehicles/87",
    headers={
        "X-Api-Key": "YOUR_API_KEY",
        "X-Tenant-Id": "TENANT_ID"
    }
)

print(response.json())
```

</CodeGroup>
