---
title: "Listar Detalles"
---

Obtén la lista de todos los detalles asociados a una factura de compra específica.

```http
GET /purchase-invoices/{id}/details
```

<Note>
  Este endpoint requiere el scope **`PurchaseInvoices.Read`**.
</Note>

### Path parameters

<ParamField path="id" type="integer" required>
  Identificador único de la factura de compra.
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request GET \
  --url "https://api.primenetcloud.com/v1/purchase-invoices/1054/details" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID" \
  --header "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch("https://api.primenetcloud.com/v1/purchase-invoices/1054/details", {
  method: "GET",
  headers: {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Tenant-Id": "TENANT_ID",
    "Content-Type": "application/json",
  },
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.primenetcloud.com/v1/purchase-invoices/1054/details",
    headers={
        "X-Api-Key": "YOUR_API_KEY",
        "X-Tenant-Id": "TENANT_ID",
    },
)

print(response.json())
```
