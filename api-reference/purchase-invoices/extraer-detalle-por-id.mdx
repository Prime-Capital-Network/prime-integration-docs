---
title: "Obtener Detalle por ID"
---

Obtén un detalle específico asociado a una factura de compra.

```http
GET /purchase-invoices/{id}/details/{detailId}
```

<Note>
  Este endpoint requiere el scope **`PurchaseInvoices.Read`**.
</Note>

### Path parameters

<ParamField path="id" type="integer" required>
  Identificador único de la factura de compra.
</ParamField>

<ParamField path="detailId" type="integer" required>
  Identificador único del detalle a consultar.
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request GET \
  --url "https://api.primenetcloud.com/v1/purchase-invoices/1054/details/1" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID" \
  --header "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch("https://api.primenetcloud.com/v1/purchase-invoices/1054/details/1", {
  method: "GET",
  headers: {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Tenant-Id": "TENANT_ID",
    "Content-Type": "application/json",
  },
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.primenetcloud.com/v1/purchase-invoices/1054/details/1",
    headers={
        "X-Api-Key": "YOUR_API_KEY",
        "X-Tenant-Id": "TENANT_ID",
    },
)

print(response.json())
```

</CodeGroup>
