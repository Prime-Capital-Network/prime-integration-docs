---
title: "Eliminar Detalle"
---

Elimina un detalle específico asociado a una factura de compra (purchase invoice).

```http
DELETE /purchase-invoices/{id}/details/{detailId}
```

<Note>
  Este endpoint requiere el scope **`PurchaseInvoices.Write`**.
</Note>

### Path parameters

<ParamField path="id" type="integer" required>
  Identificador único de la factura de compra.
</ParamField>

<ParamField path="detailId" type="integer" required>
  Identificador único del detalle a eliminar.
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request DELETE \
  --url "https://api.primenetcloud.com/v1/purchase-invoices/1054/details/1" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID"
```

```javascript JavaScript
const response = await fetch("https://api.primenetcloud.com/v1/purchase-invoices/1054/details/1", {
  method: "DELETE",
  headers: {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Tenant-Id": "TENANT_ID",
  },
});

console.log(response.status); // 204
```

```python Python
import requests

response = requests.delete(
    "https://api.primenetcloud.com/v1/purchase-invoices/1054/details/1",
    headers={
        "X-Api-Key": "YOUR_API_KEY",
        "X-Tenant-Id": "TENANT_ID",
    },
)

print(response.status_code) # 204
```

</CodeGroup>

### Response

Si la eliminación es exitosa, el endpoint retorna `204 No Content` sin body.

### Errores

<ResponseField name="400 Bad Request" type="error">
  La solicitud será rechazada si la factura ya ha sido posteada (campo `status` como `Posted`). La factura no puede ser modificada en este estado.
</ResponseField>

<ResponseField name="403 Forbidden" type="error">
  El API Key no tiene el scope `PurchaseInvoices.Write` requerido.
</ResponseField>

<ResponseField name="404 Not Found" type="error">
  No existe ninguna factura de compra con el `id` especificado, o el `detailId` no pertenece a la factura.
</ResponseField>
