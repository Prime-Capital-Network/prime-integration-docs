---
title: "Obtener Factura de Compra Por ID"
sidebarTitle: "Obtener por ID"
description: "Permite extraer una factura de compra en específico mediante REST API"
---

Obtén una factura de compra por ID en la empresa.

```http
GET /purchase-invoices/{id}
```

<Note>
  Este endpoint requiere el scope **`PurchaseInvoices.Read`**.
</Note>

### Path parameters

<ParamField path="id" type="integer" required>
  Identificador único de la factura de compra.
</ParamField>

### Example request

<CodeGroup>

```bash curl
curl --request GET \
  --url "https://api.primenetcloud.com/v1/purchase-invoices/1234" \
  --header "X-Api-Key: YOUR_API_KEY" \
  --header "X-Tenant-Id: TENANT_ID"
```

```javascript JavaScript
const response = await fetch(
  "https://api.primenetcloud.com/v1/purchase-invoices/1234",
  {
    method: "GET",
    headers: {
      "X-Api-Key": "YOUR_API_KEY",
      "X-Tenant-Id": "e3d5e6af-3ec2-49c1-92bd-9632171e9a66"
    },
  }
);
const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.primenetcloud.com/v1/purchase-invoices/1234",
    headers={
      "X-Api-Key": "pnk_abc123.s3cr3tValue",
      "X-Tenant-Id": "e3d5e6af-3ec2-49c1-92bd-9632171e9a66"
    }
)
print(response.json())
```

</CodeGroup>
