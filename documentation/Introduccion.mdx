Bienvenido a la documentación de la API de PrimeNet ERP. Prime ERP es una plataforma ERP SaaS especializada en la industria logística. Centraliza la gestión de operaciones, finanzas y procesos empresariales en una única plataforma en la nube, ayudando a las empresas a mejorar su productividad, automatizar sus procesos y tomar decisiones basadas en información en tiempo real.

Esta API te permite integrar tus sistemas con la plataforma PrimeERP para gestionar todos los recursos de tu operación logística.

## Autenticación

La API de PrimeNet utiliza **API Keys** para autenticar los requests de terceros. Cada API Key está asociada a un conjunto de permisos (scopes) que determinan a qué endpoints puedes acceder.

## Primeros pasos

<Steps>
  <Step title="Genera tu API Key">
    Ve a **Integraciones** dentro de PrimeERP y crea un nuevo registro con al menos un scope. Consulta la guía [Generar un API Key](/generar-api-key).
  </Step>
  <Step title="Autentica tus requests">
    Envía tu API Key en el header `X-Api-Key` y el ID de la compañia en el header `X-Tenant-Id` en cada solicitud. Consulta la guía de [Autenticación](/autenticacion).
  </Step>
  <Step title="Empieza a integrar">
    Explora los endpoints disponibles según los scopes de tu API Key.
  </Step>
</Steps>
